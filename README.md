#  User-Registration-Form-With-Email-Verification-Using-Nodemailer-ReactJS-Express-MongoDb


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
   
## Deployment

To deploy this project run

```bash
  npm run deploy
```
